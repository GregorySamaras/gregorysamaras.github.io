Project

Cloud & EUC Architecture

This work focused on designing and evolving end-user computing platforms that combine cloud services, identity, endpoint management, and security controls into a cohesive operating model. The emphasis was on creating environments that are secure, scalable, and practical to operate.

Context

Modern workplace platforms increasingly depend on the interaction between identity, cloud infrastructure, endpoint management, and user experience.

Technologies such as Azure Virtual Desktop, Windows 365, Microsoft 365, Entra ID, and Intune create opportunities for more flexible operating models, but also introduce architectural decisions that influence security, governance, and operational complexity.

What the work involved

  • Supporting the design and operation of Azure Virtual Desktop and Windows 365 environments.
  • Evaluating identity architecture and device management approaches across cloud platforms.
  • Contributing to modernisation initiatives involving Entra ID, Intune, and cloud-native management.
  • Reviewing how security controls, compliance requirements, and operational processes fit into platform design.
  • Supporting architectural decisions around scalability, governance, and long-term maintainability.

What mattered most

The most important lesson was that architecture is ultimately about trade-offs.

Technical decisions rarely exist in isolation. Choices around identity, endpoint management, cloud services, and security controls influence user experience, operational effort, governance requirements, and long-term sustainability.

Good architecture provides clarity and flexibility without creating unnecessary complexity. Understanding those relationships is often more valuable than understanding any individual technology.

Outcome

The work supported the development of more integrated cloud and end-user computing platforms where identity, security, governance, and operational requirements were considered together rather than independently.

It also contributed to a clearer long-term direction for platform evolution and modern workplace architecture.
